## Requisitos
- PHP 7.4 ou superior
- Servidor web nativo do PHP

## Instalação
1. Certifique-se de que o PHP está instalado em seu sistema
2. Clone ou baixe este repositório

## Execução
Para iniciar o servidor web nativo do PHP, execute:

```bash
php -S localhost:8000
```

O servidor iniciará na porta 8000.

## Uso
Acesse a aplicação no seu navegador:

```
http://localhost:8000
```
## Arquivos do Projeto

### index.php
Este arquivo é o ponto de entrada da aplicação. Ele gerencia as requisições e direciona para as páginas apropriadas.

### config.php
Contém as configurações de conexão com o banco de dados e outras variáveis de ambiente necessárias para a aplicação.
